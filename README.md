# Beauty Bot 💅

Telegram-бот для записи клиентов.
Мастера красоты подключают → клиенты записываются сами.

## Состав проекта

```
/root/beauty-bot/
├── main.py       ← Сам бот (запускать его)
├── db.py         ← База данных (SQLite)
├── config.py     ← Настройки (BOT_TOKEN)
├── bot.db        ← Создаётся автоматически при запуске
├── venv/         ← Виртуальное окружение
├── avito-ad.md   ← Текст объявления для Avito
└── README.md     ← Эта инструкция
```

## Запуск

```bash
cd /root/beauty-bot
source venv/bin/activate
export BOT_TOKEN="твой_токен_от_@BotFather"
python main.py
```

## Что нужно сделать прямо сейчас

### Шаг 1. Создать бота в Telegram
1. Открыть @BotFather
2. `/newbot` → придумать имя (например, «Запись к мастеру») → получить токен
3. Заменить `ВАШ_ТОКЕН_БОТА` в `config.py` на свой токен

### Шаг 2. Запустить
```bash
cd /root/beauty-bot
source venv/bin/activate
export BOT_TOKEN="5656...твой_токен"
python main.py
```

### Шаг 3. Пройти как мастер
1. Нажать «Я мастер»
2. Ввести имя, телефон
3. Добавить услуги (например: «Маникюр комбинированный — 1500₽ — 60 мин»)
4. Нажать «🔗 Моя ссылка» → скопировать
5. Эту ссылку кидать клиентам

### Шаг 4. Опубликовать на Avito
1. Зайти на Avito → Услуги → IT
2. Вставить текст из `avito-ad.md`
3. Указать цену 1000₽
4. Ждать клиентов
